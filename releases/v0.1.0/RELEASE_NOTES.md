# Glance v0.1.0

## Artifacts

- `Glance_0.1.0_x64_en-US.msi` - 6.43 MB
- `Glance_0.1.0_x64-setup.exe` - 4.56 MB
- `Glance-0.1.0-portable.exe` - 17.84 MB
- `GlanceCompanion-Android-0.1.0-debug.apk` - 0.97 MB

## Install

1. Install Glance on Windows using the `.exe` setup package. Use the `.msi` if you need MSI deployment.
2. Install `GlanceCompanion-Android-0.1.0-debug.apk` on the Android phone while this app is still using the development companion build.
3. Pair the phone to Glance over Bluetooth and enable the Android notification/accessibility permissions requested by the companion app.

## Notes

- Windows installers are currently unsigned, so Windows SmartScreen may warn on first install.
- The Android APK is a debug-signed companion build until a proper Android release signing key is added.
- Checksums are in `SHA256SUMS.txt` and detailed artifact metadata is in `release-manifest.json`.
