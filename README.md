# Glance releases

This repository hosts public release artifacts for Glance.

Source code is developed privately in `Stay-Umble/umble.tools.windows.glance`.

Tagged builds in the private source repository compile, sign, and publish each GitHub Release directly to this repository. Release tags and downloadable assets are therefore generated together by the source build; this repository does not rebuild or republish tags itself.

The `releases/v0.1.0` folder is retained as historical packaging output. Current release assets live on the repository's GitHub Releases page.
